# Système de Gestion des Badges Étudiants

Système de contrôle d'accès avec badges étudiants pour le BTP.

## Fonctionnalités
- Gestion des étudiants (ajout/modification/suppression)
- Contrôle d'accès par badge
- Historique des entrées
- Statistiques en temps réel

## Technologies utilisées
- HTML5
- CSS3
- JavaScript

## Comment utiliser
1. Ouvrir index.html dans un navigateur
2. Scanner un badge (ex: BADGE001, BADGE002, BADGE003)
3. Ajouter des étudiants avec leurs badges
